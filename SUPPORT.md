# Support

Little Boy's Aegis is an open-source research and education project. Community support is best-effort.

## Where to ask

- Use the affected repository's issues for bugs, setup problems, documentation gaps, and non-sensitive feature requests.
- Use pull requests for focused improvements with clear validation steps.
- Use `SECURITY.md` for vulnerabilities or anything that includes exploit details, secrets, or sensitive operational information.

## Before opening an issue

- Check the repository README and existing issues.
- Include your operating system, runtime versions, command output, logs with secrets removed, and the exact repository/component affected.
- For local deployment problems, include whether you are using Docker Compose, Kubernetes, Helm, or Terraform.

## Not supported

The project does not provide production SOC operation, incident response services, banking compliance certification, or authorization to test systems you do not own.
